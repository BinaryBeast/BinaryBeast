BinaryBeast API Ruby Gem

This Gem lets you access the Binarybeast API in an easy way. It was extracted from the Source Code of Gamkoi.com.

How to install

gem install binarybeast

bundle install

How to use

First, you should set your apikey, like this:

Binarybeast.apikey = "123456778"

anyway, we contributed the APIKey from us as a default. And you could give a tourney a direct apikey, so you don't need to bother that much about this.

After that, depending on your intention you should use

@tourney = Binarybeast::Tourney.new(:title => "Test")

to create a new Tournament on BinaryBeast or

@tourney = Binarybeast::Tourney.new(:tourneyid => "xSC21212194")

if you want to fetch an existing tournament

If you want to create a new Tournament on BinaryBeast, heres what you do:

@tourney = Binarybeast::Tourney.new(:title => "Gamkoi DevTest, :description => "Gamkoi Dev Test")

@tourney.create

your tournament is now on binary beast! Congratz!

If you want to fetch an existing Tournament on BinaryBeast, heres what you do:

@tourney = Binarybeast::Tourney.new(:tourneyid => "xSC21212194")

@tourney.load
